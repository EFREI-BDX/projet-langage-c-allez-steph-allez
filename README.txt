Première partie: Un CDataframe d’entiers
CDataframe:
Besoin de stocker des données en colonnes avec des entêtes.
Les données dans chaque colonne sont du même type.
Requiert une structure solide et flexible pour contenir un grand nombre de données et permettre des opérations comme l'ajout, la suppression et le déplacement de colonnes ou de lignes.
Les colonnes:
Structure "column" avec un titre (chaîne de caractères) et des données entières.
Utilisation de tableaux pour un accès rapide aux données.
Besoin de gérer la taille physique (nombre de cases allouées dynamiquement) et logique (nombre de valeurs insérées).
Fonctions essentielles pour les colonnes:
Créer une colonne vide à partir d'un titre.
Insérer une valeur dans une colonne avec réallocation de mémoire si nécessaire.
Libérer l'espace alloué par une colonne.
Afficher le contenu d'une colonne avec les numéros de ligne et les valeurs.

Liste des fonctions disponibles :
1. Ajouter une colonne
2. Supprimer une colonne
3. Ajouter une valeur
4. Supprimer une ligne
5. Afficher le nombre de lignes
6. Afficher le nombre de colonnes
7. Afficher les noms de colonnes
8. Afficher le CDataFrame
